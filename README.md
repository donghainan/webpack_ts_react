# 健安运力系统

## 技术栈：

`React + Antd + Axios + Redux + react-router + Typescript`

### 项目启动

`npm run dll` 启动本地打包预编译（第一次运行项目必须运行此命令，之后即可不再运行）

`npm run start` 启动本地服务

`npm run dev:build` 启动开发环境打包构建

`npm run prod:build` 启动生产环境打包构建

### 项目主要目录介绍

`build`   项目运行构建相关配置文件夹

`dll`     预编译文件夹（提升开发环境速度）

`public`  项目 index 模板文件夹

`src`     项目主要文件（重要！！！）
